Please read about the input file format requirements. Activating the "Use test-data" check-box will start an analysis using test data for the selected reference annotation. The list of available test data for the supported organisms can be found here. Or, you can take a look on the results of the hg18 test data set.

Notes: The upload must not exceed 60MB in size. Uploaded files may be in raw BAM or BED-format, or compressed using .gz, .tar.gz and .zip. The file suffix in the compressed archive must be either '.bed' or '.bam' to indicate the used format. Email is optional and allows you to receive a notification as soon as the computation has finished.

* = required field.
